#  Machine Learning Aplicado à Saúde: Predição e Análise com Streamlit

Projeto completo desenvolvido por **Pablo Riquelme** para a disciplina de *Machine Learning Aplicado à Saúde* — Faculdade Senac Recife.

O sistema foi construído em **Python** e **Streamlit**, com aplicação de técnicas de *Machine Learning supervisionadas e não supervisionadas*, utilizando datasets reais da área da saúde.

Inclui análise exploratória (EDA), predições, agrupamentos e interface interativa.

##  Estrutura
- `app.py` → Aplicação principal Streamlit
- `requirements.txt` → Dependências
- `runtime.txt` → Versão do Python (3.11.9)
- `packages.txt` → Dependências de sistema (gfortran, etc.)
- `README.md` → Explicação completa do projeto

##  Deploy
Basta enviar estes arquivos para o GitHub e conectar o repositório ao Streamlit Cloud.
